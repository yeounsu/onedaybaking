김연수, 박민재, 박정현, 유한규

1. 회원가입
 - ID 중복 확인

2. 로그인
 - ID, PW 틀렸을 경우 에러

3. 메인화면
 - 사용자는 제품을 구매하거나, 클래스 예약을 할 수 있게 함

 - 제품을 구매할 때 찜, 장바구니, 바로구매가 가능함

 - 클래스는 원데이 클래스를 목표로 하기 때문에, 사용자는 원하는 날짜에 클래스 예약을 할 수 있게 함

 - 찜 목록과 장바구니에 목록을 확인할 때 판매자에 따라 목록을 확인할 수 있게 함

 - 제품 판매나 클래스 오픈을 원하는 사용자는 상품/클래스 등록이 가능함
     -> 등록을 원하는 목록은 관리자가 승인 후 등록이 되고 메인 화면에 노출됨

 - 내 정보 페이지를 통해 정보수정, 주문내역, 클래스 예약 내역, 보유한 쿠폰을 확인할 수 있음

 - 상품을 등록한 사용자는 본인이 등록한 상품, 클래스를 확인할 수 있음

 - 상품을 구매했을 땐 구매자에 대한 구매정보를 확인하고, 클래스를 예약했을 땐 예약자에 대한 이용 인원, 시간 등을 확인할 수 있게 함

 - 구매, 예약에 대한 결제는 결제 API를 통해 결제하게 함 (테스트 결제로 이루어지며, 실제 결제 된 후 당일 11시 59분 이전에 환불)

4. 관리자
 - 관리자는 공지사항을 등록할 수 있게 함

 - 제품, 클래스 등록을 원하는 사용자의 등록 내용을 확인 후 승인함
